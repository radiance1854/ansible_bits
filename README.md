# ansible_bits

a small example of what can be done with ansible. here you can copy the created terraform artifact from my terraform_bits repo into the inventory file to target it with these playbooks

this sets up:
- linux web server with example page
- linux lnms server (partially, full setup would be considerably larger)
- linux prometheus server (basics, including a docker role first)
- windows with a few basic apps you might want via chocolatey
